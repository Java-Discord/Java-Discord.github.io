What is it?
  • If you don't know what the Java Jam is, don't worry! It's a monthly competition we organize to give you a chance to show off your Java skills and possibly win some prizes. We start by coming up with some potential topics, then spend a few days voting on the best one. Once a theme is chosen, you'll have a few weeks to work on your project. Near the end of the month, we'll close submissions and vote on the best project.
