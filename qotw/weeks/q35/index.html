  • Submission by dan1st#7327

A BubbleSort using integers could be performed like this:

public static void bubbleSort(int[] toSort){
boolean sorted=false;
for(int endIndex=toSort.length;!sorted&&endIndex>0;endIndex--){
sorted=true;
for(int j=0;j<endIndex-1;j++){
if(toSort[j]>toSort[j+1]){
int tmp=toSort[j];
toSort[j]=toSort[j+1];
toSort[j+1]=tmp;
sorted=false;
}
}
}

The outer loop iterates until the array is sorted. The inner loop checks for all all elements (except the last one) if they are bigger than the next element. If this is the case, the elements are swapped. As the highest element moves to the last index, the inner loop doesn't need to check that element in the next iteration.

  • Submission by Tahomuk#0128

A bubble sort consists of five elements:

  1. List to sort
  2. One loop with an indicator if bubbleing again is necessary
  3. Decision if bubbleing upwards or downwards
  4. Decision if sorting ascending or descending
  5. A variable swap mechanism (e.g. temporary variable)
int[] collection = new int[] {6,7,2,11,2,1}; // 1. list to sort
boolean bubble;
do {
bubble = false;
for (int i = collection.length - 1; i >= 1; i--) { // 3. bubbleing upwards
if (collection[i] < collection[i - 1]) { // 4. ascending sort
bubble = true;
int temp = collection[i]; //
collection[i] = collection[i - 1]; // 5. triangle swap mechanism
collection[i - 1] = temp; //
}
}
} while (bubble); // 2. bubble necessary?
System.out.println(Arrays.toString(collection));
